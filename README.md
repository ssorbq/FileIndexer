# Консольный индексатор папок/FileIndexer

Файл для сканирования файловой системы. Умеет находить дубликаты файлов и проверять резервные копии.

## Как запустить
1. Скачайте файл `FileIndexer.java`.
2. Откройте терминал в папке с файлом.
3. Скомпилируйте:
   ```bash
   javac FileIndexer.java
   ```
4. Запустите с нужными аргументами:
   
  Поиск дубликатов в папке:
   ```bash
   java FileIndexer D:\MyFolder
   ```
  Сравнение папки-источника и папки-резерва:
  ```bash
  java FileIndexer D:\Source D:\Backup
  ```

## Пример работы:
```
PS D:\Project> java FileIndexer D:\Test
Сканируем папку: D:\Test
Найдено файлов: 3
